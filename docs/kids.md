# 小学生向け：これってなに？

## これはなに？
IKAROS（いかろす）みたいに、**宇宙の船の向き**を変えて、
遠くの星（ここでは金星）に “ねらって” たどりつくゲームです。

## どうやってあそぶの？
1. 左のスライダーで **β_in** と **β_out** を動かします  
2. 「つぎの2週間へ（すすめる）」を押します  
3. B-plane で、点が少しずつ動きます

## なんで楕円があるの？
宇宙では、うごきが **ぴったり同じ** にならないことがあります。  
だから「つぎは、このへんに行きそうだよ」という **よそうのふくらみ** を楕円で出しています。

## 通信とでんきって？
- **でんき**：太陽にちゃんと向いているほど、たくさん作れます
- **通信**：地球のほうにうまく向けないと、お話できません

βマップを見ると、
「どこなら通信できそうか」「でんきはどれくらいか」がわかります。
