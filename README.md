# IKAROS β-GO!（HTV-GO風：ノミナル＆予測つき） v3

改善点：
- matplotlibの日本語文字化け対策（`japanize-matplotlib`）
- ライブ表示にラベルを付与（何のグラフか明確化）
- 更新頻度を調整できるようにし、軌道図は任意表示（軽量化）
- メイン表示は「軌道誤差（ノミナルとの差）」＋「β固定予測」を大きく表示

## 実行
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 操作
- 左右ボタン / ワンタッチでβ角を変更
- 高難易度：通信ウィンドウ（地球角しきい値以上）でしかβを変更できない＋燃料制限

## 注意
- 教材用の簡略モデルです（実機の精密な航法設計には使えません）。
- “燃料”は推進剤ではなく、姿勢制御/運用リソースの比喩です。
