実行:
```bash

streamlit run app.py
```


## ノミナル（計画）の定義
この教材では、ノミナル＝『理想的な初期条件（誤差ゼロ）』＋『計画β（ノミナルβ）』に従った参照（目標）軌道です。誤差は同時刻の位置差 `|r実 - rノミナル|`。
