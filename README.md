# IKAROS β-GO!（金星回廊ミッション）

「太陽光圧は小さい → 大改造ではなく“早めの微調整”が効く」ことを体験で学ぶ教材アプリです。

## コンセプト
- 宇宙機は **すでに金星へ向かう遷移軌道**に入っている（ベース設計）
- 操作は **β角（1本）**だけ
- 目標は「金星に“適切に近づく”」
  - 最近接距離を回廊 [D_MIN, D_MAX] に入れる
  - 通信ウィンドウ中は発電（β小さめ）が必要（違反はペナルティ）

## 推進モード
- SRP：太陽光圧のみ（小さなΔV）
- SPS(IES)：ソーラー電力セイル＋電気推進の併用（同じβでも効きが増える）

## 実行
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 注意
教材用の簡略モデルです（実機の精密な航法設計には使えません）。
