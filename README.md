# IKAROS β-GO!（HTV-GO風：目標軌道と予測）

要望反映版：
- ノミナル（目標）軌道：破線
- “このまま（β固定）ならどうなる？”予測：点線
- 発電量／地球角／軌道誤差（時系列）／燃料（姿勢リソース）を表示
- 制限時間終了でリザルト画面に遷移し、グラフと点数を表示
- SRPのみ（IES・事前角度指定モードなし）
- 高難易度：通信ウィンドウ時のみβ変更＋燃料制限

## 実行
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 注意
- 教材用の簡略モデルです（実機の精密な航法設計には使えません）。
- “燃料”はSRP推進の燃料ではなく、姿勢制御や運用リソースのイメージです。
